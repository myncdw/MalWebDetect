# 恶意网站检测

基于 URLhaus（abuse.ch）的浏览器用户脚本，用于在访问网页时检测已知恶意 URL，并在命中时给出本地安全提示。  
脚本完全在本地运行，不注入页面、不收集用户数据。
**URLhaus检测功能需要按下ALT+C才能触发**

## 功能

- URLhaus 实时查询（支持 API Key，可选）
- 24 小时本地缓存，自动过期清理
- 恶意页面全屏警告提示
- IDN / 视觉混淆检测（rn→m、vv→w、非拉丁字符等）
- 手动提交 URL 到 URLhaus（默认匿名，防重复）
- 自动排除本机 / 局域网地址
- 内置设置界面（API Key、缓存管理）

## 缓存说明

- 缓存键名：`urlhaus_cache`
- 仅存储在浏览器本地，由用户脚本管理器维护
- 所有缓存项默认 24 小时后清理

## 隐私

- 脚本不上传、不统计任何用户数据
- URLhaus 提交无需账号，默认匿名
- API Key 仅用于限流与可信度控制，不对第三方公开
